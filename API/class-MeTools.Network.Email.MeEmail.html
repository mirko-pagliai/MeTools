Class MeEmail

Email class.

Note that the view variables ($_viewVars) are set by MeTools\View\EmailView.

Rewrites Email.

Example:

use MeTools\Network\Email\MeEmail as Email;

$email = new Email('default');
$email->from(['me@example.com' => 'My Site'])
    ->to('you@example.com')
    ->subject('About')
    ->send('My message');
