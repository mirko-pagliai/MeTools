# url( string $path, array $options = [] )

Creates and returns the url for a thumbnail of an image or a video.

Creates and returns the url for a thumbnail of an image or a video.

To get a thumbnail, you have to use width and/or height options. To get a square thumbnail, you have to use the side option.

You can use the height option only for image files.

Note that to directly display a thumbnail, you should use the thumb() method. This method only returns the url of the thumbnail.
