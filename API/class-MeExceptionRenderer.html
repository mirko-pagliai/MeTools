Class MeExceptionRenderer

This class is used only in order to use the error views from the MeTools plugin.

In the app/Config/core.php file, you have to change the configuration of the exceptions as follows:

Configure::write('Exception', array(
    'handler' => 'ErrorHandler::handleException',
    'renderer' => 'MeTools.MeExceptionRenderer',
    'log' => true
));
