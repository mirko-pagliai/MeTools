# isAction( string|array $action, string|array $controller = NULL )

Checks if the specified action is the current action. The action name can be passed as string or array.

Checks if the specified action is the current action. The action name can be passed as string or array.

Optionally, you can also check if the specified controller is the current controller. The controller name can be passed as string or array.

Example:

$this->request->isAction('delete');

returns TRUE if the current action is delete, otherwise FALSE.

Example:

$this->request->isAction(['edit', 'delete'], 'Pages');

returns TRUE if the current action is edit or delete and if the current controller is Pages, otherwise FALSE.

$action
Action name
$controller
