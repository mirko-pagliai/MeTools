Class MeSessionComponent

Provides a way to persist client data between page requests. It acts as a wrapper for the $_SESSION as well as providing convenience methods for several $_SESSION related functions.

Rewrites SessionComponent.

You should use this component as an alias, for example:

public $components = array('Session' => array('className' => 'MeTools.MeSession'));
