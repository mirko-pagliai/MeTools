Class MeHtmlHelper

Provides functionalities for HTML code.

You should use this helper as an alias, for example:

public $helpers = array('Html' => array('className' => 'MeTools.MeHtml'));

MeHtmlHelper extends HtmlHelper.

# getCrumbs( string $separator = '/', string|array|boolean $startText = false )

Returns the breadcrumb as a sequence of separated links. Rewrites $this->Html->getCrumbs()

Returns the breadcrumb as a sequence of separated links. Rewrites $this->Html->getCrumbs()

Note that it's better to use $this->getCrumbList(), which offers better compatibility with Bootstrap

You can use Font Awesome icons for the first crumb. For example:

echo $this->Html->getCrumbs('/', array('text' => 'Homepage', 'icon' => 'icon-home'));

# getCrumbList( array $options = array(), string|array|boolean $startText = false )

Returns the breadcrumb. Rewrites $this->Html->getCrumbList()

Returns the breadcrumb. Rewrites $this->Html->getCrumbList()

You can use Font Awesome icons for the first crumb. For example:

echo $this->Html->getCrumbList(null, array('text' => 'Homepage', 'icon' => 'icon-home'));

Note that with Bootstrap 3 you don't need to set separators. Separators are automatically added in CSS through :after and content
