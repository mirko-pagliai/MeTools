Class ThumbHelper

ThumbHelper allows you to generate thumbnails of images, using Thumber.

Before using ThumbHelper, the file Plugin/MeTools/webroot/thumber.php must be copied into webroot/ (the webroot of your app).

This helper just returns the url of the thumb, then thumber.php will really create the thumb.

The thumbnails will be created in webroot/files/thumbs/ (the webroot of your app), which must be writable.

Example, in your view:

echo $this->Html->image($this->Thumb->getUrl('/path/to/image.jpg', 200));
