Class FlashComponent

Provides a way to set one-time notification messages to be displayed after processing a form or acknowledging data.

This class allows the alert(), error(), notice() and success() methods are automatically handled by the plugin and rendered dynamically using the src/Template/Element/Flash/flash.ctp template.
